# Learning Resources

A collection of links to help you on your learning journey.

## Python

- [W3Schools Python Tutorial](https://www.w3schools.com/python/) — Beginner-friendly lessons with interactive examples. Great for learning the basics step by step.

## Git & GitHub

- [GitHub Skills](https://skills.github.com/) — Hands-on courses that teach you Git and GitHub by working through real tasks in repositories.

## SSH

- [SSH Academy](https://www.ssh.com/academy/ssh) — Learn about SSH keys, secure connections, and how to use SSH with Git and remote servers.

## Claude Code & Toolkit

- [Claude Toolkit](https://github.com/hunterbrewer04/claude-toolkit) — Skills, agents, and hooks for Claude Code. Explore this to see how to customize and extend your AI assistant.
